Research Questions

What does Quality Assurance mean for web site development?
Using automated, programmable systems like Selenium to run testing on the ui, or a QA engineer will create tests that he runs daily to find issues, or there are tests that run on CI servers that will not allow you to commit code until all tests pass, or a combination of the three.
Describe the two key principles of Quality Assurance: "Fit for Purpose" and "Right First Time."
"Fit for Purpose" : The product should be suitable for the intended purpose
"Right First Time" : Mistakes should be eliminated
How do web standard relate to web quality?
Since web standards were created to promote web quality, these are intimately related
Describe what Web Accessibility means. What steps you can take to make your pages more accessible?
Web accessibility is to make sure that everyone, inclluding people with physical and situational disabilities, and socio-economic restrictions on internet bandwidth can all access your site.
