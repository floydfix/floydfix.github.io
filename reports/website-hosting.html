    Questions:
  • What is a webserver?
    A web server is one or more physical computers that contain the actual code for your website.
  • Explain why getting a domain name is not enough to put your site up on the web.
    A domain name is only a shortcut to an ip address of the webserver where you will need to host the files necessarry to run your site.
  • What service does a website hosting company provide?
    Website hosting companies provide the physical machines to store your websites files, reporting to dns servers the domain name/ip address link, and sometimes domain name registering.
  • What are the most important characteristics to look for in selecting a good web hosting company?
    Price, Hardware, Features.
