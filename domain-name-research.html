Domain Name Research report

    Questions:
  • What is an internet domain name?
    A domain name is a shortcut for finding the ip address of a website on the internet.
  • Why are domain names so important on the web?
    A domain name is easier for humans to remember than a ip address.
  • Why do domain names have to be unique?
    So that they can be unuiqely linked to thier ip address.
  • What does it mean to "register" a domain name?
    To purchase the use of that name on the internet and have it linked in domain name servers to the ip address of your sites web server.
  • If you wanted a domain name that is already taken, how could you get it?
    Purchase it from the person or company that owns it currently.
